# 📸 Instagram Clone Website (`ig-clone-website`)

A responsive front-end clone of the Instagram web interface. This project was built to practice web development fundamentals and master core Git version control workflows.

---

## 🚀 Overview

This repository contains a functional UI replica of Instagram. It covers essential layout components like the feed, navigation bar, story tray, and post cards, while providing a hands-on foundation for practicing Git concepts such as branching, merging, and commit management.

---

## ✨ Features

* **Feed & Timeline:** Clean display of posts with user headers, images, caption sections, and action icons (Like, Comment, Share).
* **Story Tray:** Top scrollable section for viewing user stories.
* **Responsive Layout:** Responsive design tailored for mobile, tablet, and desktop viewports.
* **Sidebar / Navigation:** Quick access links for Home, Search, Explore, Notifications, and Profile.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic structure for post feeds, media containers, and navigation.
* **CSS3:** Custom styling using Flexbox/Grid for layout alignment and media queries for responsiveness.
* **Git & GitHub:** Used for version control, feature branching, and commit history tracking.
